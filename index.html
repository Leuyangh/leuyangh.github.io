Computer Graphics

These are a collection of projects done with OpenGL practicing rendering images from inputs.

The first project is a heightmap renderer that takes in a black-and-white image and renders a manipulatable 3D model from it, using each pixel's brightness as its height. An example before and after. The GitHub repository for this project is here and a brief video demonstration of manipulations is here.

The second project renders a roller coaster from points that form the data to create a Catmull-Rom spline. From the spline, it's normal, binormal, and tangent are calculated and a 3D track with crossbars is rendered and the camera is moved along the center of the track according to a speed adjustable by a right-click menu in order to simulate a roller coaster. A textured world bounding box is rendered as well. The GitHub repository for this project can be found here and a brief video demonstration is here, the original data points being here.

The last project is a raytracer that takes scene data of spheres, triangles, and lights, and renders a ray-traced scene image from them. The GitHub repository for the project can be found here.

Space Miner

Space miner is a site featuring the ability to log in and play two different games. Guest users can play a single player game in which they control a starship and try to collect as many gems as possible within the time limit. Logged in users can play the single player game as well as a multiplayer game where competing ships try to collect gems and grow larger, consuming any smaller ships they hit, with the ultimate goal of controlling the stars. Unfortunately, the site is no longer being hosted but the GitHub repository for the project is here.

This was a large team project made using the waterfall development process. The single player game is made though PhaserJS which is a JavaScript framework for creating arcade style games. The multiplayer game uses NodeJS to connect and synchronize competing players.
